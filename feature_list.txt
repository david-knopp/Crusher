DESCRIPTION
  This file contains a list of planned features and currently implemented 
  features for the crusher game engine.

CURRENT FEATURES
- math
  - vec2, vec3, vec4
  - mat3, mat4
  - quat
- debug assertion macros
- type
  - non_copyable types
- high resolution timer
- static & member function delegates
- action lists
- input
  - keyboard

  - inlist
  - priority queue
- memory management
- graphics
  - 2d
  - 3d
- physics
  - 2d
  - 3d
- ai
  - fsm
  - behavior trees
  - htn planner
  - steering
  - utility
  - pathfinding
    - a*
    - navmesh
  - fuzzy logic
- action
  - common actions
- file system
- networking
  - udp
  - tcp
- meta / reflection
- scripting
  - lua
- signals
- random
- audio
- input
  - mouse
  - xbox gamepad
